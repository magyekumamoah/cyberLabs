Domain 2: Threats, Vulnerabilities, and Mitigations -22%

Study security design principles, architecture models, and secure network components including virtualization, cloud, and on-premises setups.

Domain 3: Security Architecture -18%

Hands-on implementation of cybersecurity technologies such as identity management, access control, cryptography, and endpoint security.

Domain 4: Security Operations -28%

Learn the processes for detecting, responding to, and recovering from cybersecurity incidents. Topics include threat intelligence and digital forensics basics.

Domain 5: Security Program Management and Oversight -20%

Understand security governance principles, risk management strategies, and regulatory frameworks relevant to information security.
